# うさうさ育成ごっち 🐰 — データ分析・BI・Tableau 学習アプリ

**Version:** v3.2.0（2026-06-15）／ うさうさ研修工房
**形式:** オフライン単一HTML（`usausa_ikusei_gotchi_v3.2_test100_20260615.html`）

「面白きこともなき世を面白く」── 育成ゲーム × 学習クイズ。問題に正解して 🌿草・🥕人参 を集め、うさぎを「かごのたまご → うさうさ先生🐰」へ育てながら、データ分析・BI・Tableau の基礎を反復学習できます。

---

## 1. これは何か

約100問の4択クイズを内蔵した、ブラウザだけで動く学習ツールです。サーバー不要・インストール不要・APIキー不要。ファイルをダブルクリックすれば動きます。研修の自習教材、すきま時間の復習、資格学習の足固めなどを想定しています。

設計方針は2つ。「人ではなく仕組み」（手作業に頼らず自動で回す）と「ウソつかず」（参考リンクは公式・関連・論文へ誘導し、過剰な断定をしない）。

---

## 2. 主な機能

### 学習・育成
- **📚 まなぶ**：ランダム1問。正解で 🌿草（やさしい）／🥕人参（むずかしい）を獲得。3連続正解でボーナス。知識XPが貯まると進化（たまご→ベビーうさ→こうさぎ→みならいうさ→うさうさ先生🐰）。
- **🍽️ ごはん**：集めた草・人参で「まんぷく」「きげん」を回復。人参はXPも少し増える。
- **📊 ようす**：現在の段階・XP・正答率・連続記録・首飾りコレクションを表示。サウンドON/OFF、リセットもここ。
- **🎮 10問チャレンジ**：データ分析基礎から10問。スコアに応じて 📿首飾り（🎀／🌸／💧／👑）を獲得し、うさぎに装着可能。

### 📝 100問テスト（中心機能）
- 内蔵＋カスタムから **最大100問** を出題。
- **前後ナビ**：「まえ／つぎ」ボタン＋**10×10の問題パレット**。マスは緑＝正解・赤＝不正解・グレー＝未回答で、タップで任意の問題へジャンプ。
- **○×の見直し**：回答済みの問題は読み取り専用で正解（緑）・自分の誤答（赤）を表示。
- **参考リンク**：答え合わせ時に **公式／関連／論文** の参考リンクを自動表示（カテゴリとキーワードから判定）。
- **自動一時保存＋再開**：回答ごとに自動保存。「💾 一時保存」ボタンでも明示保存。**ブラウザを閉じても、翌日でも続きから再開**できます（起動時の「ようこそ」に再開カードと通知が出ます）。
- **結果画面**：正答率・正解/不正解/未回答の内訳・**まちがえた問題一覧**（タップで該当問題へ）。

### 記録・出力
- **📥 きろく**：学習記録を **Markdown / TXT / CSV / Excel(.xls) / 印刷(PDF)** で書き出し。メモの一時保存、状態スナップショットの保存・復元も可能。

### 問題の追加（任意）
- **🧩 もんだい**：問題を **手動追加**、または **AIで生成**（APIキーは任意・未設定でも約100問で遊べます）。カスタム問題の **JSON入出力** に対応。

### 学習設計
- **間隔反復（忘却曲線）**：まちがえた問題は**翌日**、正解した問題は **1→3→7→16日** の間隔で再出題。「まなぶ」「チャレンジ」「テスト」で期限が来た問題を優先します。

## 3. 使い方（クイックスタート）

1. HTMLファイルをブラウザで開く（Chrome / Edge / Safari など）。
2. **📝 100問テスト** → スタート。
3. 選択肢をタップ → 解説と参考リンクを確認。
4. 「まえ／つぎ」やマスで移動。途中でやめてOK（自動保存）。
5. 後日また開くと「続きから再開」が出ます。

すきま時間なら **📚 まなぶ**、データ分析の腕試しなら **🎮 10問** がおすすめです。

---

## 4. データの保存について（重要）

- 進行状況は **その端末・そのブラウザの localStorage** に保存されます。
- **別の端末／別ブラウザ／シークレット（プライベート）ウィンドウ／キャッシュ削除** では引き継がれません。
- 端末を移したい場合は、🧩もんだいの **JSON書き出し**（カスタム問題）や、📥きろくの **各種エクスポート**（学習記録）をご利用ください。
- 「ようす」→「↺ はじめから」で全データ（履歴・首飾り・メモ・カスタム問題・テスト進行）を初期化できます。

保存キー：`usausa_gotchi_v3`（本体）、`usausa_gotchi_snapshot`（スナップショット）、`usausa_api_*`（AI設定）。

## 5. AIで問題を増やす（任意機能）

未設定でも内蔵問題で完結します。問題を増やしたいときだけ設定してください。

1. **🧩 もんだい → ⚙️ AI設定** でプロバイダ（OpenAI / Anthropic）・モデル・**自分のAPIキー**を入力し保存。
2. カテゴリ・難易度・問数を選び **✨ 生成する**。
3. プレビューを確認して「＋ ぜんぶ追加」。

> APIキーはこの端末の localStorage にのみ保存され、外部送信はAIプロバイダのAPI呼び出し時のみです。利用料は各プロバイダの課金に従います。共有PCでの利用や、キーの保存にはご注意ください。

## 6. カスタム問題のJSONフォーマット

`🧩 もんだい → 📥 JSON読み込み` で取り込めます。配列で渡します。

```json
[
  {
    "cat": "データ分析基礎",
    "diff": "easy",
    "q": "中央値が外れ値に強い理由は？",
    "ch": ["順位の真ん中で極端値の影響を受けにくい", "合計だから", "最大値だから", "色だから"],
    "a": 0,
    "exp": {
      "def": "中央値は順位の中央。極端な値があっても位置が動きにくい。",
      "tatoe": "1人だけ年収1億でも“真ん中の人”は変わらない🐰",
      "hito": "外れ値がいたら中央値。"
    }
  }
]
```

| フィールド | 必須 | 内容 |
|---|---|---|
| `cat` | 任意 | `データ分析基礎` / `BI` / `Tableau`（他は「その他」扱い） |
| `diff` | 任意 | `easy` または `hard`（既定 easy） |
| `q` | **必須** | 問題文 |
| `ch` | **必須** | 選択肢4つ（文字列の配列） |
| `a` | **必須** | 正解のインデックス（0〜3） |
| `exp.def` | **必須** | 定義・解説 |
| `exp.hito` | 任意 | 一言まとめ（無ければ def を流用） |
| `exp.tatoe` | 任意 | 身近なたとえ |
| `exp.detail` | 任意 | 補足解説 |

不正な要素は自動スキップされます。`cat` が「データ分析基礎」の問題は 10問チャレンジの対象にもなります。

## 8. 収録範囲（目安）

- **データ分析基礎**：尺度水準、代表値、ばらつき、相関と因果、交絡、標本/母集団、欠損・前処理、KPI、回帰・クラスタリング、仮説検定 ほか
- **BI**：BIの目的、ETL/ELT、DWH/データマート/データレイク、OLTP/OLAP、スタースキーマ、ガバナンス、RLS ほか
- **Tableau**：次元/メジャー、連続/不連続、計算フィールド、LOD（FIXED/INCLUDE/EXCLUDE）、表計算、フィルタ順序、結合/ブレンド/Union/リレーション、各種チャート ほか

## 9. トラブルシューティング

- **ボタンが反応しない／画面が真っ白** → ファイルが途中で切れている可能性。完全版を再保存してください。ブラウザのコンソール（F12）でエラー有無を確認。
- **進行が消えた** → 別ブラウザ・シークレット・キャッシュ削除が原因のことが多いです。同じブラウザで開いているか確認を。
- **音が出ない** → 「ようす」でサウンドON、ブラウザのタブを一度クリック（自動再生制限のため）。
- **AI生成が失敗する** → APIキー・モデル名・通信・利用残高を確認。未設定でも内蔵問題で学習できます。
